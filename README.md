### realtime-voting
Real-time votes for the best metal band. Made in the IGTI bootcamp using react and NodeJs.

### You need to run for the back-end:
- Npm init
- Npm install express 
- Npm Start
### You need to run for the front-end: 
- Yarn
- yarn add react-flip-move
- yarn add react-countup
- Yarn Start
